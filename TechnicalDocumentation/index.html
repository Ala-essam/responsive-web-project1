Introduction to JavaScript

JavaScript is a versatile scripting language primarily used for web development.

It can manipulate the DOM, handle events, and communicate with servers asynchronously.

JavaScript is supported by all modern browsers.

  • Client-side scripting
  • Dynamic web content
  • Interactive elements
console.log("Hello, world!");
Variables and Data Types

Variables store data that can be used and manipulated in your program.

Data types include numbers, strings, booleans, objects, and arrays.

  • var
  • let
  • const
let name = "Misha"; const age = 22;
Functions

Functions are reusable blocks of code that perform a specific task.

They can take parameters and return values.

  • Function declaration
  • Function expression
  • Arrow function
function greet(name) { return `Hello ${name}`; }
Loops

Loops allow you to execute a block of code multiple times.

Common loop types include for, while, and do-while loops.

  • for
  • while
  • do-while
for(let i = 0; i < 5; i++) { console.log(i); }
DOM Manipulation

The Document Object Model (DOM) represents the page structure.

JavaScript can dynamically change elements, attributes, and styles.

  • document.getElementById()
  • document.querySelector()
  • element.style
document.getElementById("demo").innerHTML = "Hello!";
